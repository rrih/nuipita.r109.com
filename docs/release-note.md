# ぬいぴたをリリースしました

ぬい服を買ったあとで、頭が通らなかった。

ポーチに入れてみたら、思ったよりぎゅうぎゅうだった。

そんな小さな失敗を減らしたくて、ぬいぴたを作りました。

ぬいの寸法と、服やポーチの寸法を比べて、サイズ選びの目安を確認できるWebアプリです。

## ぬいぴたでできること

### あなたのぬいを登録する

最初に、ぬいの呼び名と形を選びます。

全高、頭、胴、胴丈の4項目が分かれば、まず使い始められます。

首や腰、腕、脚などの寸法は、分かる範囲であとから追加できます。

採寸した情報は、お使いの端末内に保存されます。

### ぬい服のサイズを見る

かぶるトップス、前開きトップス、パンツ、つなぎワンピースなど、服の形を選べます。

販売ページやタグに書かれた服の寸法を入力し、登録したぬいと比べます。

頭、胴、丈など、どの部分が合いそうかを部位ごとに確認できます。

### ポーチに入るか見る

ポーチの内幅、内高、内奥行と、ぬいの大きさを比べます。

1体だけでなく、複数のぬいを並べたときの余白や向きも確認できます。

持ち歩く前に、入れ方を考える時間に使ってください。

## 判定について

表示される結果は、入力した寸法から見た目安です。

生地の伸び方、服の形、縫い方、ぬいの個体差によって、実際の着せ心地や収まり方は変わります。

購入前には、商品の実寸もあわせて確認してください。

ぴったりかどうかを機械に決めてもらうというより、気になるところを先に見つけるための道具です。

## 最初の使い方

1. [ぬいぴた](https://nuipita.r109.com/)を開く
2. 「ぬいを登録」から、あなたのぬいを1体登録する
3. 「服」または「ポーチ」を開く
4. 商品ページやポーチの内側を見ながら寸法を入力する
5. 判定結果を確認する

登録したぬいの情報は、あとから編集できます。

端末を変えるときは、「端末内データ」からバックアップを書き出して、移行先で読み込めます。

## こんなときに使ってください

- 通販で見つけた服が、あなたのぬいに合うか見たい
- 首や胴まわりがきつくないか、買う前に確認したい
- ぬいを何体かまとめてポーチに入れたい
- 以前買った服やポーチのサイズを、あとから見返したい
- 判定結果を家族や友だちに共有したい

数字を入れても、迷いがすぐゼロになるわけではありません。

それでも、届いてから気付くより、買う前に一度比べておくほうが安心です。

ぬいとのお出かけや、次の服選びに使ってもらえたらうれしいです。
